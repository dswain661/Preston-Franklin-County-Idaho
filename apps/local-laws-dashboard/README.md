# Local Laws Dashboard

Public-interest dashboard for Franklin County ordinances affecting Preston, Idaho — combined with crawled pages from official city and county websites.

## Open

**No server:** double-click `index-standalone.html`

**With server:** run `start.bat` → http://localhost:8765

## Rebuild

From repo root:

```bash
python scripts/build_all.py
```

## Views

- Overview — stats and topic breakdown
- Topics — plain-language guides (zoning, business, nuisance, etc.)
- Browse & Search — ordinances + official pages
- Preston Focus — city impact sections
- Readability — complex / high-enforcement sections
- Your Rights — civic primer
- Official Sources — 240+ crawled government pages
